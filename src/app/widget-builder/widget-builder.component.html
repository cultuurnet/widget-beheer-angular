Stel je widget samen in drie stappen:

  1. Voeg een rij toe. Hiermee bouw je de lay-out van de pagina. Je hebt keuze uit meerdere kolomstructuren. Je kan ook meerdere rijen toevoegen.
  2. Voeg een blok toe in de passende kolom. Je hebt keuze uit meerdere soorten blokken: zo kan je je resultaten op je pagina plaatsen.
  3. Configureer elke component door erop te klikken en de eigenschappen in deze rechterkolom aan te passen.

Klaar met je widget te configureren?
Klik op 'Publiceren' en plak deze code in op je eigen website.

Wil je deze pagina opmaken?
Klik op de knop 'CSS' en voeg je eigen stijlregels toe.
